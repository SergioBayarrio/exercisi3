HTTP protocol de transferencia de hipertext, ens permet fer una petició de dades i recursos a la WWW
HTML Llenguatje de Marcat de Hipertext, es el codi que s'utilitza per estructurar pagines web
URL Localitzador uniforme de recursos, direcció web d'una pagina web
TCP/IP Protocol de internet, identificador de dispositius que permet la transferencia de dades entre ells
Internet Conjunt descentralitzats de reds interconectades mitjançant un conjut de IPs
W3C World Wide Web Consortium, consorci internacional que treballa per a desenvolupar i promocionar estàndards per al World Wide Web
Mozilla Foundation Organització sense ànim de lucre dedicada a la creació de programari lliur
